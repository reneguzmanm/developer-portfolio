# Definition of Done

Un caso de estudio se considera terminado cuando cumple con los siguientes criterios:

- [ ] Hero.
- [ ] Contexto del proyecto.
- [ ] Mockup.
- [ ] Flujo de la solución.
- [ ] Mi participación.
- [ ] Decisiones de ingeniería.
- [ ] Tecnologías utilizadas.
- [ ] Diseño responsive.
- [ ] Build sin errores.
- [ ] Revisión final de textos.
